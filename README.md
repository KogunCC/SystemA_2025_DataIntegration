reactのディレクトリに移動
```
cd SystemA_2025_DataIntegration
```

ディレクトリ上で実行
```
npm start
```
APIサーバ接続
```
    cd SystemA_2025_DataIntegration/api-server
    node server.js
```

DB接続

```
    cd SystemA_2025_DataIntegration/api-server/
```
.env.exampleを複製します。
```
    cp .env.example .env
```
その後、ファイル内の指示に従ってパスワード等を書き換えてください。
